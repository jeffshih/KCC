# Color classifie

### Dependency 

cuda > 9.0  

keras = 2.2.4  

tensorflow = 1.11.0  

opencv = 3.4.3  


### Install procedure

sudo pip install tensorflow-gpu   

sudo pip install keras  

sudo pip install opencv-python  


### Usage

python Demo.py -i <imagePath> 

or

download test dataset and run

python Demo.py
